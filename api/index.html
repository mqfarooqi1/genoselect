    API reference

    Prediction models

    genoselect.GBLUP

    Bases: BaseEstimator, RegressorMixin

    Genomic BLUP regressor.

    Fits GBLUP on a 0/1/2 marker matrix, estimating variance components by REML. The equivalent ridge-regression marker effects are derived so the model can predict breeding values for unseen genotypes.

    Attributes (set after fit)

    h2_ : float Estimated genomic heritability Vu / (Vu + Ve). Vu_, Ve_, lambda_ : float Genetic and residual variances and their ratio. gebv_ : ndarray Genomic estimated breeding values for the training individuals. marker_effects_ : ndarray RR-BLUP marker effects (for out-of-sample prediction).

    genoselect.StackedEnsemble

    Bases: BaseEstimator, RegressorMixin

    Super-learner that combines base models with non-negative weights.

    Out-of-fold predictions from an inner cross-validation are stacked, and the combination weights are found by non-negative least squares (then normalised to sum to one), following the super-learner of van der Laan et al. (2007). Base models are then refit on the full training data.

    genoselect.available_models()

    Names of the genomic-prediction models genoselect can run.

    genoselect.make_model(name, random_state=None)

    Construct a fresh, unfitted estimator by name.

    All estimators follow the scikit-learn fit(X, y) / predict(X) API, where X is a 0/1/2 marker matrix.

    genoselect.gblup

    GBLUP by REML (Endelman 2011 spectral method), scikit-learn compatible.

    GBLUP

    Bases: BaseEstimator, RegressorMixin

    Genomic BLUP regressor.

    Fits GBLUP on a 0/1/2 marker matrix, estimating variance components by REML. The equivalent ridge-regression marker effects are derived so the model can predict breeding values for unseen genotypes.

    Attributes (set after fit)

    h2_ : float Estimated genomic heritability Vu / (Vu + Ve). Vu_, Ve_, lambda_ : float Genetic and residual variances and their ratio. gebv_ : ndarray Genomic estimated breeding values for the training individuals. marker_effects_ : ndarray RR-BLUP marker effects (for out-of-sample prediction).

    reml_solve(y, K, X=None)

    Solve the mixed model y = Xb + g + e, g ~ N(0, K s2u), e ~ N(0, I s2e).

    Variance components are estimated by REML using the spectral approach of Endelman (2011) — the profiled restricted log-likelihood is minimised over the single ratio lambda = s2e / s2u. Returns a dict with Vu, Ve, h2, beta, gebv, lambda, Hinv and the residuals (y - Xb).

    gblup(y, geno=None, K=None, X=None, min_maf=0.0)

    Functional interface mirroring the R GSbench::gblup.

    With geno it returns a fitted :class:GBLUP (which can predict new genotypes); with only K it returns the REML solution dict (no out-of-sample prediction).

    genoselect.reml_solve(y, K, X=None)

    Solve the mixed model y = Xb + g + e, g ~ N(0, K s2u), e ~ N(0, I s2e).

    Variance components are estimated by REML using the spectral approach of Endelman (2011) — the profiled restricted log-likelihood is minimised over the single ratio lambda = s2e / s2u. Returns a dict with Vu, Ve, h2, beta, gebv, lambda, Hinv and the residuals (y - Xb).

    Relationship matrix

    genoselect.vanraden_grm(geno, min_maf=0.0)

    VanRaden (2008) genomic relationship matrix from 0/1/2 dosages.

    G = W Wᵀ / (2 Σ p(1-p)), with W = M - 2p the centred marker matrix.

    genoselect.centre_markers(geno, min_maf=0.0)

    Centre markers by 2p and return the VanRaden scaling.

    Returns (W, p, c, keep) where W is the centred marker matrix (n x m_kept), p the allele frequencies of the kept markers, c the VanRaden normaliser 2 * sum p(1-p), and keep the boolean column mask.

    genoselect.benchmark

    Benchmark several genomic-prediction models on the same data.

    benchmark(geno, y, models=None, k=5, reps=1, scheme='kfold', groups=None, random_state=None)

    Cross-validate all (or selected) models and return a comparable summary.

    Defaults to every model except the (slow, nested) ensemble. Use CVResult.summary() for a tidy table sorted by predictive ability.

    Per-fold predictive abilities for one or more models.

    GWAS

    genoselect.gwas

    Single-marker genome-wide association (GWAS) with structure correction.

    gwas(geno, y, marker_map=None, n_pc=0, covariates=None)

    Single-marker additive association scan.

    Each marker is tested for association with y after regressing out an intercept, optional covariates, and (optionally) the first n_pc genotype principal components to correct for population structure.

    Parameters:

    Name Type Description Default
    geno array(n, m)

    Marker matrix coded 0/1/2.

    		 required
    y array(n)

    Phenotype (missing values are dropped).

    		 required
    marker_map DataFrame or dict

    Per-marker annotation; recognised columns marker, chrom, pos are carried through to the result (enabling position-aware Manhattan plots).

    		 None
    n_pc int

    Number of genotype PCs to include as covariates.

    		 0
    covariates array(n, k)

    Extra fixed covariates.

    		 None

    Returns:

    Type Description
    DataFrame

    One row per marker with columns index [, marker, chrom, pos], effect, se, t, p and log10p.

    manhattan(result, alpha=0.05, ax=None, colors=('#2c7fb8', '#8fbcd4'), title='GWAS Manhattan')

    Manhattan plot from a :func:gwas result (position-aware if chrom).

    qq_plot(result, ax=None, title='QQ plot')

    QQ plot of observed vs expected -log10(p) from a :func:gwas result.

    Reading data

    genoselect.read_vcf(path)

    Read a (optionally gzipped) VCF into dosages = number of ALT alleles.

    Biallelic diploid sites are used; multiallelic sites are skipped.

    genoselect.read_plink(prefix)

    Read a PLINK 1 binary fileset (.bed/.bim/.fam).

    Dosage is the number of copies of the .bim A1 allele (0/1/2, NaN missing). Only SNP-major .bed files are supported (the PLINK default).

    genoselect.read_hapmap(path)

    Read a HapMap file into dosages = copies of the second listed allele.

    genoselect.GenotypeData dataclass

    Genotypes plus identifiers.

    geno is (n_samples, n_markers) allele dosage (0/1/2, NaN = missing).

    Simulation and QC

    genoselect.simulate_population(n=200, m=1000, n_qtl=50, h2=0.5, seed=None)

    Simulate n individuals genotyped at m biallelic markers.

    n_qtl markers are causal with Normal effects; the genetic values are scaled to unit variance and Normal noise is added to give the target narrow-sense heritability h2.

    genoselect.qc_markers(geno, maf=0.05, max_missing=0.1, impute=True)

    Filter markers by minor-allele frequency and missingness.

    Returns (filtered_geno, keep_mask). With impute=True remaining missing values are mean-imputed.

    genoselect.impute_markers(geno)

    Mean-impute missing genotypes (NaN) column-wise (by 2p).
